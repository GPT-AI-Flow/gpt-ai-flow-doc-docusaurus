# 模拟对话-面试

## 提示词

我希望你扮演面试官的角色，我会扮演应聘者的角色，我现在面试一个 { } 的职位。你只需要回答我的问题，不需要一次性写下所有对话。你需要像面试官一样一个一个问题地向我提问，并等待我的回答。
